# Prompts

Esta pasta reúne todos os prompts utilizados durante o desenvolvimento do projeto **IA Aplicada à Gestão da Produção Industrial com NotebookLM**.

Os prompts foram elaborados para explorar diferentes funcionalidades da ferramenta, gerar respostas técnicas, organizar o conhecimento e apoiar os estudos sobre Gestão da Produção Industrial.

---

## Conteúdo da Pasta

| Arquivo | Descrição |
|----------|-----------|
| `prompts.md` | Coleção dos prompts utilizados para gerar resumos, perguntas, respostas, comparações e guias de estudo no NotebookLM. |

---

## Objetivo

Centralizar todos os prompts utilizados no projeto para facilitar a reutilização, documentação e evolução dos estudos.
