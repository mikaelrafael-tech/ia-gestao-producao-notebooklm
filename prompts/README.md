# Prompts Utilizados no NotebookLM

## Prompts

Este documento reúne os principais prompts utilizados durante o desenvolvimento do projeto **IA Aplicada à Gestão da Produção Industrial com NotebookLM**. Eles podem ser reutilizados para estudos, pesquisa e construção de novos projetos.

---

## 1. Resumo Geral

Elabore um resumo completo de todas as fontes adicionadas ao notebook, destacando os principais conceitos, metodologias, ferramentas e boas práticas relacionadas à Gestão da Produção Industrial.

---

## 2. Guia para Supervisor de Produção

Crie um guia completo para um Supervisor de Produção Industrial contendo:

- principais responsabilidades;
- competências técnicas;
- competências comportamentais;
- indicadores utilizados;
- ferramentas de melhoria contínua;
- exemplos práticos de aplicação.

## 3. Lean Manufacturing

- consequências;
- como evitá-los;
- boas práticas de implantação.
