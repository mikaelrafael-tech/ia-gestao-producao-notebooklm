# IA Aplicada à Gestão da Produção Industrial com NotebookLM

## Sobre o projeto

Este projeto foi desenvolvido durante o curso da DIO com o objetivo de explorar o uso da Inteligência Artificial aplicada à Gestão da Produção Industrial utilizando o NotebookLM.

O foco foi criar um material de estudo profissional reunindo conceitos de Lean Manufacturing, PDCA, 5S, OEE e KPIs, utilizando recursos de IA para organização do conhecimento, geração de resumos, mapas mentais e perguntas estratégicas.

---

## Objetivos

- Aplicar IA na organização do conhecimento
- Criar material de estudo profissional
- Demonstrar o uso do NotebookLM
- Construir um portfólio para GitHub e LinkedIn
